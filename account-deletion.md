# GymCheck 계정·데이터 삭제 안내

GymCheck(개발자: 유월필름 / 이준)는 사용자가 언제든지 계정과 모든 데이터를 영구 삭제할 수 있도록 지원합니다.

## 방법 1 — 앱에서 직접 삭제 (즉시)

1. GymCheck 앱 실행 후 로그인
2. **프로필** 탭으로 이동
3. **계정 삭제** 버튼 탭
4. 확인하면 계정과 모든 운동·인바디 데이터가 **즉시 영구 삭제**됩니다.

## 삭제되는 데이터

계정 삭제 시 다음 데이터가 **모두 영구 삭제**되며 복구할 수 없습니다.

- 계정 정보(이메일, 사용자 식별자)
- 운동 기록(날짜·부위)
- 인바디 측정 기록(체중·골격근량·체지방률)
